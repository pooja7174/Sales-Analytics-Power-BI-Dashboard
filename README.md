# Sales-Analytics-Power-BI-Dashboard

Classic Models Business Intelligence Project

An end-to-end Power BI project built using relational sales data to analyze revenue, profitability, product demand, and operational performance for a classic automotive company. An interactive Power BI dashboard designed to analyze automotive sales performance, profitability, and product demand across multiple product lines and years. This report provides strategic insights into revenue drivers, customer trends, and product-level performance within a classic and vintage car business model.
